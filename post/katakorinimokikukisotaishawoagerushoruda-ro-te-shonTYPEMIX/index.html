肩こりにも効く！基礎代謝を上げるショルダーローテーション「TYPE MIX」

2016.6.23 スポーツ remove_red_eye549

■連載／ メンズビューティー通信

広背筋トレーニングの最後を飾るのは、肩を回す動きのMIX。肩周り全体を動かすため、肩こりの解消には特に効果がある。また、動きが一番大きいため、これまでの「 TYPE Y 」「 TYPE T 」「 TYPE W 」を行った後にやることで、緊張した筋肉をほぐす効果も。これで「背中で語る男」の準備は全て整った！

広背筋トレーニングの総仕上げ　TYPE MIX

吉田「TYPE MIXは肩甲骨を回す動きが入ります。よって、肩周りの筋肉がほぐれ、肩こり解消には一番効果のあるトレーニングです。また、動きをつけることで、広背筋も収縮しやすいため効率よく鍛えることができます。」

吉田「まずは、TYPE Yのポーズをとりますが、このとき腕の幅をYのときよりやや狭くして、全身で"I"の形を作ってください。

吉田「次に、手の平を内向きに返しながら肘を曲げていきます。このとき、肩甲骨を寄せるイメージを持つと、広背筋がギュッと収縮します。」

吉田「肘を曲げきり、これ以上広背筋が寄らない位置まで持ってきたら2秒キープします。肩が凝っていたり、パソコンワークのせいで背中が張っていたりすると、ちょっと痛いかもしれません。ですが、続けることで確実に筋肉がほぐれていき、なおかつ形の良い後ろ姿が形成できるのでチャレンジしてみてください。2秒キープしたら、再び手の平を返しながら、またIの形に戻ります。」

NG!

吉田「筋肉を収縮させようとして、背中を反るのはNGです。これは腰を痛めやすいうえ、広背筋にはあまり効いていないので注意してください。顎を上げてしまうとこの体勢になりやすいので、顎を引き、常に床を見続けるのがポイントです。」

吉田トレーナーの今日の教訓

・今までのトレーニングの仕上げに行うべし！
・腕を曲げるだけでなく、肩甲骨を寄せるイメージを持つべし！
・絶対に背中を反るべからず！

フィジカルトレーナー
吉田輝幸

1975年生まれ。国士舘大学体育学部卒業。大学卒業後に本格的にパーソナルトレーナーの道を進むため、渡米。帰国後はさまざまなスポーツ、ジャンルのアスリートのフィジカルトレーナーとして活躍。正しい身体の使い方、正しい動きによって最高のパフォーマンスを引き出す「吉田コアパフォーマンスメソッド」を提唱。現在は、トップアスリートやモデル、アーティストなどが通うトレーニングジム、「PCP」のフィジカルトレーナー＆ディレクターを務める。また、多くのフィットネスクラブや専門学校でのパーソナルトレーナー養成やトレーニングマシーン開発補助、テレビや雑誌などさまざまなメディアでも活動している。
